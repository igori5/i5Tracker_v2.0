# i5Tracker_v2.0
素人が適当に作りました。  
SlimeVRでトラッカーを作るときに引いた基盤です。  
WemosD1は技適が怪しいためESP-WROOM02にしてます。  
ピン番号合わせたのでWemosD1用のプログラムで動きます。  
RSTとIO0引き出してますが私はボタンとジャンパで書き込みモードにしました。  
充電管理基板とMPU6500ボードと電池ボックスは中華の汎用品なのでアリエクとかで適当に探してください。  
色々適当なので信用しないように(重要）
